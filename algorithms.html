Boyer-Moore Voting Algorithm

This Algorithm is mainly used for finding Majority Element. A majority Element is any element in an array with size n that appears more than n/2 times.

Sliding Window Technique

The Sliding window is a problem-solving technique for problems that involve arrays/lists. These problems are easy to solve using a brute force approach in O(n^2) or O(n^3). Using the 'sliding window' technique, we can reduce the time complexity to O(n). ---Stack Overflow.
